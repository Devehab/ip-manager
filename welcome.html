Welcome to IP Manager

A powerful tool to manage and monitor your IP address

First Step: Pin the Extension

1. Click the Extensions icon in your Chrome toolbar
2. Click the pin icon next to IP Manager
3. Now you can easily access IP Manager from your toolbar

IP Address Display

View your public IP address with its type (IPv4/IPv6). The address is automatically updated every 5 minutes to ensure accuracy.

Port Checker

Check port status on local and public networks. Supports custom host checking, with easy-to-read results and one-click URL copying.

Network Test

Verify your internet connection status with one click. Tests multiple servers to ensure accurate results.

IP History Management

Track IP changes with advanced filtering, sorting, and export capabilities. Includes timestamps and IP types for comprehensive tracking.

Quick Tips

  • Click the copy icon to copy any IP address or port URL to your clipboard
  • Use the search bar in history to filter specific IP addresses
  • Sort history by time to track changes chronologically
  • Export your IP history as a log file for record keeping
  • Use custom host checking to test specific domains or IPs

How to Enable Notifications

If you are not receiving notifications, follow these steps to enable them:

Enable Notifications in Chrome

  • Go to chrome://settings/content/notifications and allow notifications for Chrome.

Enable Notifications on Windows

  • Go to Settings > System > Notifications & actions.
  • Ensure that notifications are enabled for Google Chrome.

Enable Notifications on macOS

  • Go to System Preferences > Notifications.
  • Find Google Chrome and enable notifications.

Disable "Do Not Disturb" on macOS

  • Open Control Center from the top-right corner.
  • Click on "Focus" and ensure "Do Not Disturb" is off.
