Welcome to IP Manager

A powerful tool to manage and monitor your IP address

First Step: Pin the Extension

1. Click the Extensions icon in your Chrome toolbar
2. Click the pin icon next to IP Manager
3. Now you can easily access IP Manager from your toolbar

IP Address Display

View your public IP address with its type (IPv4/IPv6). The address is automatically updated every 5 minutes to ensure accuracy.

Port Checker

Check the status of any port on your local machine or public IP. Supports both local and public port scanning.

Network Test

Verify your internet connection status with one click. Tests multiple servers to ensure accurate results.

Change History

Track all IP address changes with timestamps. Keeps the last 10 changes for easy reference.

Quick Tips

  • Click the copy icon to copy the IP address to your clipboard
  • Use the refresh button to manually update your IP address
  • Monitor changes in the history log at the bottom of the window
